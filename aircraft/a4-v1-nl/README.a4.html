Model description and updates:

2003/05/22 - First release: v1-nl

* The flight model is based largely on data from Nelson's book (see
  the aircraft.dat file for details).

* Model includes:
  - A SAS has been added in roll, pitch and yaw which probably closely
    approximates the CSS model of the real A4.
  - Includes nonlinear aero effects like stall
  - Flaps are modeled and actuated via the '[' and ']' keys
  - Gear is modeled and actuated via the 'g' key
